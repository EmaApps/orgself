# orgself

Create beautiful journal websites with self-tracking. Using org-mode. Based on [ema](https://github.com/srid/ema).

## Running

```bash
bin/run
```

Or <kbd>Ctrl+Shift+B</kbd> in VSCode.

## Upgrading ema

Open `flake.nix` and change `ema.url` to point to a new revision.
